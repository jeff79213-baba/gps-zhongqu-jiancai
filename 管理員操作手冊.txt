     中區建材行 GPS 打卡系統 — 管理員完整操作手冊

一、系統網址與登入
════════════════════════════════════

系統網址：https://opencode-sk.web.app

打開網址後會進入登入畫面。

角色與帳號：
  管理員：帳號「管理員」密碼「admin123」
  會計：帳號「會計」密碼「acc123」
  員工：由管理員新增帳號

⚠️ 重要提醒：
  - 請妥善保管密碼，忘記後可從「🏢 系統設定」頁面修改
  - 建議首次登入後立即修改密碼


二、功能總覽
════════════════════════════════════

📍 GPS 定位打卡（上班/下班）
📊 即時統計（今日狀態、工時、遲到、加班）
📋 歷史紀錄查詢（按月篩選）
📅 月曆檢視（出勤一目瞭然）
📊 出勤總覽（全部員工出勤狀況，管理員/會計）
💰 薪資試算（自動計算加班費、全勤獎金，管理員/會計）
⚙️ 管理後台（員工管理、補登打卡，管理員限定）
🏢 系統設定（打卡範圍、時間、休息日、獎金，管理員限定）
📥 匯出 Excel / JSON
🔒 管理員密碼修改
🔑 首次登入強制改密

四、設定打卡範圍（第一次使用必做）
════════════════════════════════════

員工必須在公司附近才能打卡，因此第一次使用時必須設定公司位置。

操作步驟：
  1. 點擊下方「🏢 設定」分頁
  2. 找到「📍 打卡範圍設定」區塊
  3. 點擊「📍 取得目前位置作為公司」
  4. 輸入打卡範圍（預設 500 公尺）
  5. 點擊「💾 儲存範圍」


五、設定上下班時間
════════════════════════════════════

  1. 在「🏢 設定」分頁找到「⏰ 上下班時間」
  2. 輸入上班時間（預設 09:00）和下班時間（預設 18:00）
  3. 點擊「💾 儲存上下班時間」


六、設定休息時間
════════════════════════════════════

  1. 在「🏢 設定」分頁找到「🍽️ 休息時間設定」
  2. 設定午休開始和結束時間
  3. 選擇是否扣除休息時間（工時計算用）
  4. 點擊「💾 儲存休息時間」


七、設定全勤獎金
════════════════════════════════════

  1. 在「🏢 設定」分頁找到「🏆 全勤獎金設定」
  2. 輸入全勤獎金金額（0 = 不啟用）
  3. 條件：當月無遲到 + 無缺勤
  4. 點擊「💾 儲存全勤獎金」


八、設定休息日
════════════════════════════════════

  1. 在「🏢 設定」分頁找到「📅 休息日設定」
  2. 勾選公司固定休息日
  3. 點擊「💾 儲存休息日」


九、新增員工帳號
════════════════════════════════════

  1. 點擊「⚙️ 管理」分頁
  2. 找到「👤 新增員工」區塊
  3. 輸入員工姓名、選擇角色、輸入基本月薪、設定密碼
  4. 點擊「➕ 新增員工」

⚠️ 重要：
  - 帳號密碼只會顯示一次，請務必記錄
  - 員工首次登入後需自行更改密碼


十、補登打卡
════════════════════════════════════

  1. 點擊「⚙️ 管理」分頁
  2. 找到「📋 補登打卡」區塊
  3. 選擇員工、日期、類型（上班/下班）、時間
  4. 點擊「📋 補登」

⚠️ 補登的紀錄會標記「補登」方便辨識


十一、查看薪資試算
════════════════════════════════════

  1. 點擊「💰 薪資」分頁
  2. 選擇月份
  3. 系統自動計算每位員工的：
     - 基本月薪
     - 加班費（依勞基法費率）
     - 全勤獎金
     - 應發金額


十二、匯出資料
════════════════════════════════════

  - 📥 匯出 Excel：下載 CSV 檔案，可用 Excel 開啟
  - 📄 匯出 JSON：下載 JSON 檔案，可用於系統對接


十三、員工打卡流程
════════════════════════════════════

  1. 打開系統網址
  2. 輸入帳號密碼登入（首次需更改密碼）
  3. 進入「📍 打卡」頁面
  4. 系統自動讀取 GPS 位置
  5. 點擊「上班打卡」或「下班打卡」按鈕
  6. 打卡成功後今日紀錄會顯示在下方


十四、常見問題
════════════════════════════════════

Q：員工忘記密碼怎麼辦？
A：管理員可刪除帳號後重新建立，打卡紀錄不受影響。

Q：員工在範圍內但系統顯示不在範圍？
A：可能是 GPS 信號弱，請到空曠處或窗邊重試。

Q：休息日員工打卡會怎樣？
A：休息日可以正常打卡，月曆上不會顯示為缺勤。

Q：管理員忘記密碼怎麼辦？
A：請聯繫技術人員從後台資料庫直接修改。
