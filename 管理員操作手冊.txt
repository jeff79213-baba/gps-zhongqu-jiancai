     中區建材行 GPS 打卡系統 — 管理員完整操作手冊

二、功能總覽
════════════════════════════════════

本系統包含以下功能：

  📍 GPS 定位打卡（上班/下班）
  📊 即時統計（今日狀態、工時、遲到、加班）
  📋 歷史紀錄查詢（按月篩選）
  📅 月曆檢視（出勤一目瞭然）
  ⚙️ 管理後台（出勤統計、員工管理）
  📥 匯出 Excel / JSON
  🔒 管理員密碼修改
  📅 休息日設定（例假日不需打卡）
  🔢 員工自動編號
  🔑 首次登入強制改密


三、設定打卡範圍（第一次使用必做）
════════════════════════════════════

員工必須在公司附近才能打卡，因此第一次使用時必須設定公司位置。

操作步驟：
  1. 點擊下方「👤 設定」分頁
  2. 找到「🏢 打卡範圍設定」區塊
  3. 點擊「📍 取得目前位置作為公司」
     → 系統會自動抓取您目前的 GPS 位置
  4. 輸入打卡範圍（預設 500 公尺）
  5. 點擊「💾 儲存範圍」

⚠️ 注意：
  - 建議在公司內部操作此步驟
  - 範圍外打卡會顯示「不在打卡範圍內」無法打卡


四、設定上下班時間
════════════════════════════════════

  1. 在「👤 設定」分頁找到「⏰ 上下班時間」
  2. 輸入上班時間（預設 09:00）和下班時間（預設 18:00）
  3. 點擊「💾 儲存上下班時間」

遲到判定：
  - 上班時間之後打卡 → 標記為「遲到」（紅色）
  - 上班時間之前打卡 → 標記為「準時」（綠色）


五、設定休息日（新增功能）
════════════════════════════════════

設定公司固定休息日，這些天員工不需打卡，月曆上不會顯示為缺勤。

操作步驟：
  1. 在「👤 設定」分頁找到「📅 休息日設定」
  2. 勾選公司固定休息日（可多選）
  3. 點擊「💾 儲存休息日」

範例：
  - 週六、週日休息 → 勾選「週六」和「週日」
  - 只休週日 → 只勾選「週日」
  - 排休制 → 不勾選任何一天


六、設定員工編號規則
════════════════════════════════════

可設定的項目：
  - 編號前綴：例如 A、B、EMP
  - 起始編號：例如 1、100
  - 編號位數：補零用，3位=001

範例：前綴「A」、起始「1」、位數「3」→ A001, A002, A003...

操作步驟：
  1. 在「👤 設定」分頁找到「🔢 員工編號規則」
  2. 設定前綴、起始編號、位數
  3. 預覽欄位會自動更新
  4. 點擊「💾 儲存編號規則」


七、新增員工帳號
════════════════════════════════════

操作步驟：
  1. 點擊「⚙️ 管理」分頁
  2. 找到「👤 新增員工」區塊
  3. 輸入員工姓名和初始密碼
  4. 點擊「➕ 新增員工」
  5. 系統會顯示帳號和密碼，請立即記錄並告知員工

⚠️ 重要：
  - 帳號密碼只會顯示一次，請務必記錄
  - 員工首次登入後需自行更改密碼


八、修改管理員密碼（新增功能）
════════════════════════════════════

  1. 在「👤 設定」分頁找到「🔒 修改管理員密碼」
  2. 輸入目前密碼
  3. 輸入新密碼（至少4個字元）
  4. 再輸入一次新密碼確認
  5. 點擊「🔒 修改密碼」
  6. 下次登入請使用新密碼


九、查看出勤紀錄
════════════════════════════════════

  - 「📋 紀錄」頁面：查看個人歷史打卡紀錄，可按月篩選
  - 「📅 月曆」頁面：月曆式檢視，綠點=準時，紅點=遲到，灰字=缺勤
  - 「⚙️ 管理」頁面：查看全部員工出勤紀錄，可篩選月份和員工


十、匯出資料
════════════════════════════════════

  - 📥 匯出 Excel：下載 CSV 檔案，可用 Excel 開啟
  - 📄 匯出 JSON：下載 JSON 檔案，可用於系統對接

匯出內容包含：員工編號、姓名、日期、時間、類型、GPS 座標


十一、員工打卡流程（供轉知員工用）
════════════════════════════════════

  1. 打開系統網址
  2. 輸入帳號密碼登入（首次需更改密碼）
  3. 進入「📍 打卡」頁面
  4. 系統自動讀取 GPS 位置
  5. 點擊「上班打卡」或「下班打卡」按鈕
  6. 打卡成功後今日紀錄會顯示在下方

⚠️ 注意：
  - 必須開啟手機 GPS 定位
  - 必須在公司打卡範圍內
  - 每天只能打一次上班、一次下班


十三、常見問題
════════════════════════════════════

Q：員工忘記密碼怎麼辦？
A：管理員可刪除帳號後重新建立，打卡紀錄不受影響。

Q：員工在範圍內但系統顯示不在範圍？
A：可能是 GPS 信號弱，請到空曠處或窗邊重試。

Q：休息日員工打卡會怎樣？
A：休息日可以正常打卡，月曆上不會顯示為缺勤。

Q：管理員忘記密碼怎麼辦？
A：請聯繫技術人員從後台資料庫直接修改。
